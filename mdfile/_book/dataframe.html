Chapter 6 dataframe

6.1 Dataframe from column vectors

A data frame is a collection of columns, each of which corresponds to an observed variable (in the statistical sense, not the programming sense). If your data is already organized into columns, then it’s easy to build a data frame. The data.frame function can construct a data frame from vectors, where each vector is one observed variable. Suppose you have two numeric predictor variables, one categorical predictor variable, and one response variable. The data.frame function can create a data frame from your vectors:

names <- c("kedar","shyam","mohan","geeta","seema","priya")
ages <- c(23,22,21,22,23,23)
height <- c(165,170,167,160,164,166)
has_specs <- c("yes","no","no","yes","no","yes")
data.frame(names,ages)
##   names ages
## 1 kedar   23
## 2 shyam   22
## 3 mohan   21
## 4 geeta   22
## 5 seema   23
## 6 priya   23
data.frame(names,height)
##   names height
## 1 kedar    165
## 2 shyam    170
## 3 mohan    167
## 4 geeta    160
## 5 seema    164
## 6 priya    166
data.frame(ages,height)
##   ages height
## 1   23    165
## 2   22    170
## 3   21    167
## 4   22    160
## 5   23    164
## 6   23    166
data.frame(height,has_specs)
##   height has_specs
## 1    165       yes
## 2    170        no
## 3    167        no
## 4    160       yes
## 5    164        no
## 6    166       yes
data.frame(names,ages,height,has_specs)
##   names ages height has_specs
## 1 kedar   23    165       yes
## 2 shyam   22    170        no
## 3 mohan   21    167        no
## 4 geeta   22    160       yes
## 5 seema   23    164        no
## 6 priya   23    166       yes
data.frame(Name=names,Age=ages,Height=height,Glasses=has_specs)
##    Name Age Height Glasses
## 1 kedar  23    165     yes
## 2 shyam  22    170      no
## 3 mohan  21    167      no
## 4 geeta  22    160     yes
## 5 seema  23    164      no
## 6 priya  23    166     yes

## 
## [[4]]

6.2 Dataframe from rows vectors

Data often arrives as a collection of observations. Each observation is a record or tuple that contains several values, one for each observed variable. Such data is organized by initializing a Data Frame from Row Data observation, not by variable. In other words, you are given rows one at a time rather than columns one at a time.

obs <- list(
  c("Name"="kedar","Age"=23,"Height"=165,"Glasses"="yes"),
  c("Name"="shyam","Age"=22,"Height"=170,"Glasses"="no"),
  c("Name"="mohan","Age"=21,"Height"=167,"Glasses"="no")
)
obs[[1]]
##    Name     Age  Height Glasses 
## "kedar"    "23"   "165"   "yes"
rbind(obs[[1]],obs[[2]])
##      Name    Age  Height Glasses
## [1,] "kedar" "23" "165"  "yes"  
## [2,] "shyam" "22" "170"  "no"

We want to bind together every observation, not just the first two, so we tap into the vector processing of R. The do.call function will expand obs into one, long argument list and call rbind with that long argument list:

 do.call(rbind,obs)
##      Name    Age  Height Glasses
## [1,] "kedar" "23" "165"  "yes"  
## [2,] "shyam" "22" "170"  "no"   
## [3,] "mohan" "21" "167"  "no"

6.10 combine on specific column

born<-rbind(data.frame("name"="mohan",
                 "place_b"="shirdi",
                 "year_b"=1959),
            data.frame("name"="girija",
                 "place_b"="pune",
                 "year_b"=1950)
            
)

dead<-rbind(data.frame("name"="mohan",
                 "place_d"="shirdi",
                 "year_d"=2018),
            data.frame("name"="girija",
                 "place_d"="pune",
                 "year_d"=2000)
)
